## 🚀 Deploy Your First Client (45 Minutes)

### Quick Path: Use `QUICK_DEPLOY.md`

**Step 1: Create Telegram Bots** (5 min)
- @BotFather → /newbot (5 times)
- Save tokens

**Step 2: Extract Agent Files** (5 min)
- Extract to ~/.openclaw/workspace/agents

**Step 3: Backup Config** (2 min)
- Save existing OpenClaw config

**Step 4: Update OpenClaw Config** (10 min)
- Add Telegram section + agent registration
- Use interactive or manual method

**Step 5: Validate & Restart** (5 min)
- `openclaw doctor` → Should pass
- `openclaw gateway restart`

**Step 6: Deploy via Morgan** (20 min)
- Message @[client]-morgan on Telegram
- Morgan asks about business
- You approve team
- Morgan deploys all agents

**Result:** All 5 agents active, ready to work
